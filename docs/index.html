Gomen

Imagine a network of agents playing a two-player game in time, changing their strategies in time. Gomen is a small project aiming to determine if information-theoretic tools such as mutual information or transfer entropy can allows us to infer the game, strategy rule and topology of the graph given only the time series of the agents' strategies.

To answer this question, we need a number of components

  1. A specification of a two-player game (Game)
  2. An arena, i.e. a graph, on which the game is played (Arena)
  3. A scheme for updating player strategies (Scheme)
  4. Information-theoretic methods (Information)
